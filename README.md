# 🚗 Vehicle Parts Management System

An Enterprise-Level Vehicle Parts Management System developed using **ASP.NET Core MVC + Entity Framework Core + SQL Server**.

This system helps manage vehicle spare parts inventory, suppliers, customers, sales invoices, role-based access, and purchase workflows efficiently.

---

## 🔐 Authentication & Authorization
- Customer Registration & Login
- Session-Based Authentication
- Role-Based Authorization
- Admin / Staff / Customer Roles
- Secure Access Control

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/VehiclePartsManagementSystem.git
```

---

## 2️⃣ Open Project

Open solution file in:
```bash
Visual Studio 2022
```

---

## 3️⃣ Configure Database

Update connection string inside:

```bash
appsettings.json
```

---

## 4️⃣ Run Migrations

Open Package Manager Console:

```bash
Update-Database
```

---

## 5️⃣ Run Project

Press:

```bash
F5
```
